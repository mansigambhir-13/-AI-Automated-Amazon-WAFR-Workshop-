- `import copy`, `import time`, `from decimal import Decimal`
- `import boto3`, `from botocore.exceptions import ClientError`
- `from boto3.dynamodb.conditions import Key, Attr`

**Helper functions** (module-level, before the class):

1. `_python_to_dynamodb(obj)` â€” Recursively convert Python dict/list/float for DynamoDB storage. `float` becomes `Decimal(str(float_val))`. Pass through `str`, `int`, `bool`, `None`, `Decimal` unchanged. Handle nested dicts and lists.

2. `_dynamodb_to_python(obj)` â€” Reverse: `Decimal` becomes `int(obj)` if `obj == int(obj)` else `float(obj)`. Recurse into dicts and lists.

3. `_compute_ttl_365d()` â€” Returns `int(time.time()) + (365 * 24 * 60 * 60)`.

**DynamoDBReviewStorage class:**

Constructor `__init__(self, sessions_table, review_sessions_table, users_table, s3_bucket, region)`:
- Create `boto3.resource("dynamodb", region_name=region)` and store Table references for all three tables.
- Create `boto3.client("s3", region_name=region)` for S3 operations.
- Store `s3_bucket`. Set `S3_OVERFLOW_THRESHOLD = 300 * 1024` (300KB) as class constant.
- Log initialization with table names.

**ABC method implementations:**

1. `save_session(self, session_data)`:
   - Extract `session_id` from `session_data` (raise ValueError if missing).
   - Compute `expires_at` via `_compute_ttl_365d()`.
   - Store session metadata row in `wafr-review-sessions` with `item_id='SESSION'`. Include fields: session_id, item_id, status (default "ACTIVE"), created_at (default `datetime.utcnow().isoformat()`), updated_at, transcript_answers_count, summary, assessment_summary, expires_at. Apply `_python_to_dynamodb()` before `put_item`.
   - For each item in `session_data.get("items", [])`, store a separate row in `wafr-review-sessions` with `item_id=review_item["review_id"]`, including all item fields plus session_id and expires_at. Apply `_python_to_dynamodb()`.

2. `load_session(self, session_id)`:
   - Query `wafr-review-sessions` with `KeyConditionExpression=Key("session_id").eq(session_id)`.
   - Separate the `item_id='SESSION'` row (metadata) from other rows (review items).
   - If no SESSION row found, return None.
   - Apply `_dynamodb_to_python()` to all items.
   - Reconstruct session dict: metadata fields + `"items": [review_items]`.
   - Return the reconstructed dict.

3. `delete_session(self, session_id)`:
   - Query all items for the session_id from `wafr-review-sessions`.
   - Batch delete all items using `batch_writer()`.
   - Return True if items were deleted, False if none found.

4. `list_sessions(self, status=None, limit=100)`:
   - If `status` provided: query `status-created_at-index` GSI with `KeyConditionExpression=Key("status").eq(status)` and `FilterExpression=Attr("item_id").eq("SESSION")`, `ScanIndexForward=False`, `Limit=limit * 2` (over-fetch because filter reduces results). Handle pagination if `LastEvaluatedKey` present.
   - If no status: `scan` with `FilterExpression=Attr("item_id").eq("SESSION")`, `Limit=limit`.
   - Apply `_dynamodb_to_python()` to each returned item.
   - Return list of session metadata dicts (without review items â€” list is summary only).

5. `update_item(self, session_id, review_id, item_data)`:
   - Apply `_python_to_dynamodb()` to `item_data`.
   - Ensure `session_id` and `item_id=review_id` are in the item.
   - Add `expires_at` via `_compute_ttl_365d()`.
   - Use `put_item` on `wafr-review-sessions` to overwrite the specific item row.
   - Return True on success, False on error (catch ClientError).

6. `save_validation_record(self, record)`:
   - Extract `session_id` (raise ValueError if missing).
   - Store in `wafr-review-sessions` with `item_id='VALIDATION'`.
   - Apply `_python_to_dynamodb()`, add `expires_at`.

7. `load_validation_record(self, session_id)`:
   - `get_item` from `wafr-review-sessions` with `Key={"session_id": session_id, "item_id": "VALIDATION"}`.
   - Apply `_dynamodb_to_python()`. Return item or None.

**Additional methods (not in ABC but needed for Phase 2):**

8. `save_pipeline_results(self, session_id, results)`:
   - Deep copy `results`. Navigate to `results["steps"]["wa_workload"]["review"]` (if it exists) and delete `report_base64` key. Set `report_base64_stripped = True` as a flag.
   - Serialize cleaned results to JSON string via `json.dumps(results_clean, default=str)`.
   - If JSON string length > `S3_OVERFLOW_THRESHOLD`: upload to S3 at `dynamo-overflow/pipeline_results/{session_id}.json`, store DynamoDB item in `wafr-sessions` with `pipeline_results_s3_key` pointer and `pipeline_results_json` set to None.
   - Else: store JSON string directly in `wafr-sessions` item as `pipeline_results_json` attribute.
   - Always include `session_id`, `created_at` (required sort key â€” use `datetime.utcnow().isoformat()`), and `expires_at` in the `wafr-sessions` put_item.

9. `load_pipeline_results(self, session_id)`:
   - Query `wafr-sessions` with `KeyConditionExpression=Key("session_id").eq(session_id)`, `ScanIndexForward=False`, `Limit=1` to get most recent item.
   - If item has `pipeline_results_s3_key`: fetch from S3, parse JSON.
   - Elif item has `pipeline_results_json`: parse JSON string.
   - Else: return None.
   - Return parsed dict.

10. `save_transcript(self, session_id, transcript_text)`:
    - Always store transcript in S3 at `dynamo-overflow/transcripts/{session_id}.txt` (per locked decision: transcripts always in S3).
    - Store `transcript_s3_key` reference in `wafr-sessions` item via `update_item` (UpdateExpression to set the key without overwriting other attributes).

11. `sync_user_profile(self, user_id, email, groups=None, display_name=None)`:
    - Write to `wafr-users` table. Item: `user_id` (PK), `email`, `groups` (list of strings, default []), `display_name`, `created_at`, `updated_at`.
    - Use `put_item` with the full item (upsert semantics). Apply `_python_to_dynamodb()`.

Wrap all DynamoDB and S3 calls in try/except ClientError with appropriate logging. Never let exceptions silently pass â€” log at ERROR level and re-raise or return appropriate failure values.
